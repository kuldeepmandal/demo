Flexbox vs Grid

Flexbox is designed for one-dimensional layouts (either row or column). It's best for aligning items along a single axis and is perfect for navigation bars, card layouts, and centering content.

Grid is designed for two-dimensional layouts (rows and columns simultaneously). It's ideal for complex page layouts, photo galleries, and when you need precise control over both dimensions.

This portfolio uses Grid for the project cards layout to create a responsive multi-column design that adapts automatically.
